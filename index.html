Exam Generator

Abstract

This report introduces ExamGenerator, an AI-powered web application designed to automate the creation of exam questions and answers from document inputs such as PDFs, Word, and text files. The tool utilizes natural language processing (NLP) to convert educational materials into comprehensive exams, including multiple-choice, true/false, and multiple-selection questions. The app also supports batch processing for handling multiple documents and provides compatibility with various learning management systems (LMS) through export options. This report outlines the motivation behind ExamGenerator, its system architecture, proposed approaches, and related work in the field.
